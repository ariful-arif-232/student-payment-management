# Student Payment and Finance Management System

This is a console-based C programming project for managing student payment and finance records.

## Project Description

The Student Payment and Finance Management System helps to manage student accounts, installment dues, payments, clearance lists, and section allocation.

This project is developed using C programming language and basic data structures.

## Features

- Add account for one student
- Add accounts for multiple students
- Add first installment due
- Add final installment due
- Payment for one student
- Payment for multiple students
- Show student list and due amount
- Show clearance list
- Section application
- Section based student list

## Data Structures Used

### Doubly Linked List
Used to store student registration number and due amount.

### Queue
Used to store students who have cleared their payment.

### Array
Used to store section-wise student lists.

## Tech Stack

- C Programming
- Data Structure
- Doubly Linked List
- Queue
- Array

## How to Run

1. Open the project in Code::Blocks, Dev-C++, or VS Code.
2. Compile the C file.
3. Run the program.
